execute

Request a runtime adaptation

Enact a change

Used to adapt the exemplar at runtime

Request Body schema: application/json
required

Apply an adaptation

server_number
integer

Sets the number of servers.

dimmer_factor
number

Set the proportion of requests served with optional content (number between 0 and 1).
